

Configure the user name.
$ git config --global user.name "AbhishekGandre";

Configure the user email.
$ git config --global user.email "gandreabhishek@gmail.com";

Config list
$ git config --list

Clone the repository
$ git clone https://github.com/Git

Check Status
git status
